# Menu cocktails

Page web statique pour la cremaillere.

## Ouvrir en local

Ouvre `index.html` dans un navigateur. Les photos et les icones viennent d'internet, donc il faut une connexion.

## Hebergement gratuit simple

### Option la plus rapide: Netlify Drop

1. Va sur https://app.netlify.com/drop
2. Glisse-depose le dossier `Menu`
3. Netlify donne une URL publique, accessible depuis telephone et tablette.

### Option propre avec GitHub Pages

1. Cree un depot GitHub.
2. Ajoute `index.html`, `styles.css` et `script.js`.
3. Dans GitHub, va dans `Settings > Pages`.
4. Choisis la branche principale et le dossier racine.
5. GitHub donne une URL publique.

Netlify Drop est le plus simple pour un evenement ponctuel.
